# Linkly – Secure URL Shortener with Click Analytics
Linkly is a secure URL shortening service that allows users to generate short URLs, track clicks, and view analytics. Each user has authenticated access to their links and click data.

---

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, REST APIs
- **Frontend:** React.js, TailwindCSS
- **Database:** MySQL (JPA/Hibernate)
- **Tools:** Docker, Maven

---

## Key Features
- User registration, login, and JWT-based authentication  
- Authenticated short URL generation
- HTTP redirection for short URLs
- Click tracking with analytics dashboard
- Role-based access control
- Intuitive frontend interface for link management
